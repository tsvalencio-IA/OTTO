# Athos: Guardião dos Portais — V17

Versão focada em transformar a base V16 em um jogo mais verdadeiro, mantendo o coração do projeto:

- Athos 3D real com `athos.glb`;
- AR nativo pelo `model-viewer`;
- Brincar Livre AR com câmera real + Three.js;
- controles limpos no gameplay, sem Quiz/Falar dentro da tela de jogo;
- fase lateral estilo plataforma;
- checkpoints;
- inimigos;
- plataformas móveis;
- dano real;
- morte/respawn em checkpoint;
- lava tirando pontos;
- buraco causando morte;
- cristais com propósito para abrir o portal;
- ícone novo infantil e seguro.

## Como testar

Suba tudo em um repositório GitHub Pages e abra pelo celular em HTTPS.

1. Teste o Lobby e o AR Nativo.
2. Entre em Jogar Fases.
3. Use o D-pad para andar.
4. Use A para pular.
5. Use B para poder.
6. Use X para alternar mini/normal/gigante.
7. Use Y para abaixar.
8. Ative checkpoint.
9. Pule inimigos ou use fase com cuidado.
10. Entre no portal somente depois de coletar cristais e cumprir objetivos.

## Observação

Quiz e Falar ficam somente no Lobby, conforme solicitado.
